# Redeliste

Eine Echtzeit-Multiuser-Redeliste.

![][screenshot]

[screenshot]: public/redeliste.png

## Dependencies

- [Perl][Perl] 5.20
- [Mojolicious][Mojo] 8.24

[Perl]: https://www.perl.org/get.html
[Mojo]: https://mojolicious.org/

## Install

    $ cpanm -n --installdeps .

## Run

    $ hypnotoad redeliste
